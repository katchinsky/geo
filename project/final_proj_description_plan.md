**Описание финального проекта**

**Цель проекта (бизнес постановка задачи)**

Оценить доступность крытых бассейнов и иных круглогодичных водных объектов (indoor water facilities) для жителей Белграда. Выделить районы с наименьшей обеспеченностью. 

**Математическая постановка задачи**

Таргет: доля жилых гексов Н3 в каждом районе Белграда, для которых расстояние до ближайшего indoor water facility больше километра. 

Гипотеза: круглогодичные водные объекты распределены неравномерно и сосредоточены вокруг спортивных центров и рекреационных зон, а в плотной жилой застройке доступность относительно низкая. Неравенство доступа коррелирует с обеспеченностью населения 

**Источники данных**

* точки интереса: бассейны, спа, public bath, спортивные центры с OSM   
* административные границы районов Белграда с OSM  
* карта распределения населения GHSL  
* данные о высотности зданий из Global Building Atlas или данные о зданиях с OSM   
* данные о доходах населения по районам RZS Serbia [https://www.stat.gov.rs/en-US/](https://www.stat.gov.rs/en-US/)  
* данные о стоимости жилья c nekretine.rs

**Методы**

* Загрузка и фильтрация indoor water facilities из OSM  
* Выделение жилой застройки по тегам OSM (building=house/apartments/residential/dormitory)  
* Классификация зданий с общим OSM тегом building=yes с помощью supervised модели, обученной на зданиях с конкретными тегами (с геометрическими и контекстными фичами)  
* Построение гексагонов H3 и пересечение и с жилой застройкой  
* Агрегирование населения из GHSL по каждому гексагону  
* Расчёт расстояния до ближайшего бассейна и числа объектов в 1000 м буферной зоне  
* Расчёт медианной стоимости жилья по гексагонам на основе данных с Nekretnine.rs  
* Привязка данных о доходах по муниципалитетам к гексагональной сетке  
* Стандартизация показателей доступности (z-score) для выявления районов с наименьшей обеспеченностью, сравнение распределений показателей доступности между районами (квантильное разбиение)  
* Оценка пространственной автокорреляции показателей доступности с использованием индекса Moran’s I и локальных индикаторов пространственной ассоциации (LISA)  
* Расчет коэффициента корреляции Спирмана между показателями доступности и доходами / стоимостью жилья

**Краткий план реализации проекта**

* Загрузить данные по водным объектов, отфильтровать outdoor-only;  
* Загрузить административные границы Белграда;  
* Построить гексагоны, пересечь с жилой застройкой, отфильтровать гексы без жилых зданий  
* Рассчитать показатели доступности: расстояние от центра каждого гекса до ближайшего indoor water facility, расчёт количества объектов в радиусе 1000 м;  
* Агрегировать население в гексагоны и оценка доступности на жителя;  
* Сопоставить муниципальных данных о доходах (RZS) с районами;  
* Загрузить данных о ценах аренды или продажи жилья (Nekretnine.rs) и расчет медианной стоимости в каждом гексе (через H3 clustering).  
* Провести статистический анализ неравенства и пространственной автокорреляции показателей доступности  
* Визуализировать результаты на карте
